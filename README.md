# Adoption-Site

This project was an exercise to create an adoption site while practicing ES6 modules and building data in a JSON file.

## How to run this project
* Use npm to install http-server in your terminal:
```sh
npm install -g http-server
```
* Run the server
```sh
hs -p 9999
```
* Open chrome and navigate to:
```
localhost:9999
```
